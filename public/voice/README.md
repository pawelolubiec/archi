# Slide narration audio

Place the ten generated narration files in this folder using exactly these names:

- `slide01.mp3`
- `slide02.mp3`
- `slide03.mp3`
- `slide04.mp3`
- `slide05.mp3`
- `slide06.mp3`
- `slide07.mp3`
- `slide08.mp3`
- `slide09.mp3`
- `slide10.mp3`

When narration is enabled, the presentation automatically starts the matching file whenever the viewer changes slides. Missing files are handled without interrupting slide navigation.
